About

This project is a combination of useful utilities for interacting with SVG elements using Angular. It is a work in progress, so I will be updating this regularly and adding more items. To use this in your page, you just need to add the file to your page and add a dependency to the svg-interactive module.

angular.module('myModule', ['svg-interactive']);

Below are examples of the items in the module. Use them separately or combine them together for even more possibilities.

Zoomable

The zoomable directive is used for creating an SVG area that allows zooming in and out using the mouse wheel. The zoom is centered on the user's cursor and can be enabled or disabled by setting the zoomable attribute to either true of false.

To control the rate of zoom, set the zoom-rate attribute to a number (default is 1.1).

It's required to set an initial view box on the SVG area as it will manipulate that to produce the zoom effect.


    <svg xmlns="http://www.w3.org/2000/svg" zoomable="true" zoom-rate="1.1"
         width="400" height="300" viewBox="0 0 1000 1000">
        <rect x="100" y="100" height="150" width="150"></rect>
        <circle cx="250" cy="500" r="100"></circle>
        <line x1="750" y1="150" x2="900" y2="750" stroke-width="10" stroke="black"></line>
    </svg>

Pannable

The pannable directive is used for creating an SVG area that allows panning around using the mouse to click and drag.

It's required to set an initial view box on the SVG area as it will manipulate that to produce the zoom effect.


    <svg xmlns="http://www.w3.org/2000/svg" zoomable="true" zoom-rate="1.1"
    width="400" height="300" viewBox="0 0 1000 1000">
    <rect x="100" y="100" height="150" width="150"></rect>
    <circle cx="250" cy="500" r="100"></circle>
    <line x1="750" y1="150" x2="900" y2="750" stroke-width="10" stroke="black"></line>
    </svg>

Bringing them Together


        <svg xmlns="http://www.w3.org/2000/svg" zoomable="true" zoom-rate="1.1" pannable

        <circle cx="250" cy="500" r="100"></circle>
